
  # Portfolio Website Design

  This is a code bundle for Portfolio Website Design. The original project is available at https://www.figma.com/design/YftQBVhAzfEVZ9bOwftufX/Portfolio-Website-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Markdown in project content

  Project long-form text supports Markdown so you can use lists, bold text, and paragraphs.

  Common examples:

  ```markdown
  This is a paragraph with **bold text**.

  - Bullet item one
  - Bullet item two

  1. Numbered item one
  2. Numbered item two
  ```

  Where to update it:

  - Update the Excel/CSV documents in `src/profile info/generated` (content flows into the site).

  Official Markdown renderer docs:

  - https://remarkjs.github.io/react-markdown/
